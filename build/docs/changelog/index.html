0.4.0 (2020-11-10)#

For new Ledokku users#

Ledokku from now on will run on port 80, instead of 4000.

For existing Ledokku users#

If you want to have same port behaviour as decribed ☝️ , you will need to run these commands :

dokku proxy:ports-add ledokku http:80:4000
dokku proxy:ports-remove ledokku http:4000:4000

Features#

  • client: pre populate db name on creation (#235) (65c8c0c)
  • add app port mapping management (#231) (ad4dd4e)
  • allow to install custom ledokku version via bootstrap script (#229) (7088aee)
  • configure number of users allowed per server (#233) (57e199b)
  • ledokku listen on port 80 instead of 4000 on production (#230) (e0424a7)
